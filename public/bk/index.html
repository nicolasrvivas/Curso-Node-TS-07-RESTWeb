Sobre este proyecto

Este web server fue desarrollado en Node.js + TypeScript como parte de mi primer proyecto práctico siguiendo un curso de Udemy. El objetivo fue construir una base sólida y moderna para servicios web, aplicando buenas prácticas desde el inicio: tipado estricto, arquitectura limpia y un pipeline simple de calidad.

Tecnologías y enfoque

  • Stack: Node.js, TypeScript, (Express/Fastify), dotenv.
  • Arquitectura: rutas → controladores → servicios → capa de datos; middlewares para validación, manejo de errores y logging.
  • Calidad: ESLint + Prettier, scripts de npm para lint/build/test.
  • Documentación: endpoints descritos con OpenAPI/Swagger.
  • Pruebas: (Jest/Vitest) para unidades críticas.
  • Ops: variables de entorno, configuración por ambiente y opción de despliegue con Docker.

Qué resuelve

Provee endpoints REST básicos listos para ampliar (salud, recursos CRUD y utilidades), con manejo consistente de respuestas y errores, pensado para crecer sin perder mantenibilidad.

Próximos pasos

Autenticación JWT, pruebas de integración, cacheo selectivo, métricas (Prometheus) y CI/CD.

Si quieres revisar o proponer mejoras, ¡bienvenido! Este proyecto nació para aprender haciendo y seguir iterando con disciplina técnica.
